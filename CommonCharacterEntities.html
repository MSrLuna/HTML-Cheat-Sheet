Ejemplo de Entidades Numéricas HTML

En este ejemplo, utilizaremos algunas entidades numéricas HTML para mostrar caracteres especiales:

  • ": Comillas dobles (")
  • &: Ampersand (&)
  • <: Signo menor que (<)
  • >: Signo mayor que (>)
  • @: Símbolo de arroba (@)
  • €: Símbolo de Euro (€)
  • •: Viñeta (•)
  • ™: Símbolo de marca registrada (™)
  • £: Símbolo de Libra esterlina (£)
  •  : Espacio en blanco sin interrupciones ( )
  • ©: Símbolo de copyright (©)
